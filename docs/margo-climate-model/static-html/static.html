Now we ask MARGO what the optimal approach is to limit warming to 2.5 degrees celcius. Notice that overwhelmingly the algorithm suggesets that we should focus on reducing emissions, but some amount of carbon dioxide removal and geoengineering is helpful too.

If we want to keep warming to 1.5 degrees, the amount of emissions reduction needs to increase sharply.

Use the controls to see what MARGO suggests is the best path to limiting warming to a specific temperature. You can also change what exactly MARGO optimizes for. Optimizing for net-benefit considers the cost-benefit analysis. Optimizing for temperature is optimizing only for limiting the warming under a certain temperature, while optimizing for adaptiv_temp is a slightly looser constraint that allows for adaptation to higher temperature.
